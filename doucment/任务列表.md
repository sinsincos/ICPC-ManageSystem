# 开发计划计划 （verson 1.0）
## 1 登录注册模块(index)
### 1.1 登录 (index)
#### 1.1.1 前端js逻辑
- 检查当前登录状态（getState.php）
  - 已登录：调用jump(my.html, index)
- 登录按钮触发事件（userLogin.php）
  - 表单检查
    - 正确：提交，后端返回结果
       - 正确：调用jump(my.html, index)
       - 错误：提示错误信息
    - 错误：提示错误信息
  
#### 1.1.2 后端接口
- getState.php
  - 传入：NULL
  - 返回：
    - state：0 / 1 （未登录 / 已登录）
- userLogin.php
  - 传入：
    - username
    - password （md5加密后）
  - 返回：
    - state：0 / 1 （登录失败 / 登录成功）
    - log (备注：登录信息 / 错误信息)
